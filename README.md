# Base Talk

- Reveal.js 2.6.1
- Separate markdown files
- Solarized theme
- Tweaked syntax highlighting

### Requirements

- npm
- grunt-cli

### Usage

```
$ cp -r ./this ./a_talk
$ cd ./a_talk
$ npm install
$ grunt serve
$ $BROWSER http://localhost:8000
$ $EDITOR index.html  # change title, add sections
$ $EDITOR slides/*.ms # add content
```
